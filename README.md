# Tradutor de Artigos Técnicos com AzureAI

Este projeto tem como objetivo traduzir artigos técnicos automaticamente utilizando a API do **Azure OpenAI**.  

O código foi desenvolvido durante o bootcamp da **DIO** para a certificação **IA 102**.

---

## Funcionalidades

- Traduz artigos técnicos de um idioma para outro.
- Utiliza o modelo de linguagem do Azure OpenAI.
- Permite adaptar o prompt para diferentes estilos de tradução.
- Mantém a integridade do conteúdo técnico.
